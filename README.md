# Финальный проект - Аналитик данных

Данный репозиторий содержит финальный проект, выполненный в рамках курса **«Аналитик данных» от Academica Courses**.

Проект охватывает полный цикл работы с данными: от **извлечения и обработки данных до анализа, визуализации и подготовки аналитических отчетов**. В ходе выполнения проекта были использованы различные инструменты аналитики, позволяющие анализировать транзакционные данные, поведение клиентов и ключевые бизнес-показатели.

Основная цель проекта - **получить практический опыт работы с аналитическими инструментами и выявить бизнес-инсайты**, связанные с активностью клиентов и динамикой продаж.

# Структура проекта

Проект состоит из нескольких блоков, каждый из которых направлен на решение различных аналитических задач.

---

# SQL блок

В данном разделе проводится анализ клиентских и транзакционных данных с использованием SQL.

Основные задачи:

- поиск клиентов с **непрерывной историей транзакций за год**;
- расчет **среднего чека**, средней суммы покупок и количества операций;
- анализ показателей **в разрезе месяцев**:
  - средняя сумма чека;
  - среднее количество операций;
  - количество активных клиентов;
  - доля операций и оборота;
- анализ **гендерного распределения клиентов** и структуры их расходов;
- сегментация клиентов по **возрастным группам** и анализ транзакционной активности.

[Посмотреть SQL файл](SQL/SQL_part.sql)

---

# Excel блок

В данном блоке выполняется анализ данных с использованием инструментов Excel.

Основные задачи:

- объединение данных из нескольких источников;
- построение **сводных таблиц** по неделям и территориям;
- определение **топ-3 территорий**:
  - по доле в общем товарообороте;
  - по обороту на склад за последнюю неделю;
- построение **графиков динамики товарооборота**;
- выполнение дополнительных аналитических заданий.

---

# Python блок

В этом разделе используется Python для анализа и обработки данных.

Основные задачи:

- загрузка и предварительная обработка данных;
- исследовательский анализ данных (**EDA**);
- расчет статистических показателей;
- визуализация данных;
- ответы на аналитические вопросы на основе датасета.

[Посмотреть Python файл](Python/Python_part.ipynb)

---

# Power BI блок

В данном блоке создается интерактивный аналитический отчет.

Основные задачи:

- разработка **дашбордов**;
- визуализация ключевых бизнес-показателей;
- анализ динамики метрик;
- подготовка итогового отчета в форматах **PBIX и PDF**.

---

# Тестовое задание

В данном разделе выполняется отдельное аналитическое задание, включающее:

- ответы на теоретические вопросы;
- решение практических задач;
- использование SQL или Python для анализа данных;
- подготовку результатов в формате документа.

---

# Итог

В рамках проекта были применены основные инструменты аналитики данных: **SQL, Excel, Python и Power BI**.

Проект демонстрирует навыки:

- работы с базами данных;
- анализа транзакционных и клиентских данных;
- обработки данных и расчета метрик;
- построения визуализаций и аналитических отчетов.

Полученные результаты позволяют выявлять **закономерности в поведении клиентов и динамике продаж**, что может быть использовано для поддержки принятия бизнес-решений.
